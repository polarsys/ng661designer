- Project folder shall be Y:\A661p2\A661p2_simulator

	+ Y:\A661p2Dependencies\GnuWin32
	+ Y:\A661p2Dependencies\lib-ext\freetype-2.3.5-1-bin
	+ Y:\A661p2Dependencies\lib-ext\ShivaVG
